# Run and deploy the portfolio

This is a frontend-only Vite portfolio that can be deployed directly to Netlify or any static host.

## Run Locally

**Prerequisites:** Node.js


1. Install dependencies:
   `npm install`
2. Run the app:
   `npm run dev`

## Build For Deploy

1. Create the production build:
   `npm run build`
2. Deploy the generated `dist` folder to Netlify.
